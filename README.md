# tomox-bot
send orders to tomox rpc

This bot helps to increase liquidity for the market, not for taking profit

## Build
```
./build.sh
```

## Run
 ```
 cp .env.sample .env 
 # update .env file 
 
 # see all binaries at /build, copy binary corresponding your os
 cp build/tomox-bot-linux-amd64 ./bot
 
 # run bot 
 ./bot
 ```
 Argument: 
 - StartNonce
